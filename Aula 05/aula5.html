Criação de variáveis em JavaScript

Variáveis são espaços reservados na memória do cumputador onde é possível guardar qualquer informação.

Em JavaScript utilizamos as palavras var,let ou const para criar variáveis.

Existem algumas regras para nomear variáveis em JavaScript (regras e boas práticas):

  1. as variáveis podem começar com letras, $ e _;
  2. as variáveis não podem começar com números;
  3. você pode usar letras e números para suas variáveis (desde que não comece com o número);
  4. o sistema aceita uso de acentos e símbolos (mas não é uma boa prática);
  5. não pode ter espaços em branco na variável e o sistema diferencia maiúsculas de minúsculas;
  6. não podem ser palavras reservadas, como function, var, let, const, etc;
  7. Como boa prática, deve-se usar o estilo CamelCase para nomear suas variáveis. Nesse estilo, diferentes palavras são juntadas, sendo a primeira palavra iniciada em minúscula e as seguintes iniciadas em maiúsculas, tudo junto;

Tipos de variáveis

As variáveis no JavaScript podem ser de vários tipos:

  • number = variáveis numérias, podem seer qualquer número, inteiro, gracionário, positivo, negativo, + infinito, - infinito;
  • string = variáveis que recebem texto;
  • bolean = variáveis classificadas em verdadeiras e falsas;
  • null = tratado como um objeto no JavaScript, null é uma variável com valor nulo;
  • undefined = variável não definida;
  • array = variáveis que armazenam vários tipos de dados ao mesmo tempo;
  • function = são blocos de comados definidos previamente;
